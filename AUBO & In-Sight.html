핵심 설계 개념

단일 로봇을 사용하여 입고(프로세스 A)와 출고(프로세스 B)를 모두 수행합니다. 입고 시에는 "Blind(좌표 고정)" 방식을 사용하고, 출고 시에는 "Vision(가변 좌표)" 방식을 사용하여 비뚤어진 제품을 정렬하며 이송합니다.

메인 루프 (Main Loop)

IF (수량 < 최대) → 프로세스 A
ELSE → 프로세스 B

↓
프로세스 A (블라인드 적재)
  1. 픽업 위치 이동 (컨베이어)
  2. 그리퍼 ON (집기)
  3. 팔레트 위치 계산 (인덱스)
  4. 적재 (비전 미사용)
  5. 카운트 증가
↓
프로세스 B (비전 정렬)
  1. 카메라 촬영 위치로 이동
  2. TCP 트리거 → 좌표 수신 (X, Y, Rz)
  3. 손목을 회전(Rz)하여 집기
  4. 배출 위치로 이동 (손목 0°)
  5. 카운트 초기화
프로세스 B 서브 루틴 핵심 코드입니다.
-- 통신 및 데이터 파싱 로직
local vision_ip = "192.168.1.100"
local tcp_client = tcp_connect(vision_ip, 3000)

-- 촬영 요청
tcp_send(tcp_client, "Trigger")
sleep(0.5)

-- 데이터 수신 "X,Y,Angle"
local recv_str = tcp_recv(tcp_client)
local data = string_split(recv_str, ",")

-- 좌표 변환
local v_x = tonumber(data[1])
local v_y = tonumber(data[2])
local v_rz_deg = tonumber(data[3])
local v_rz_rad = v_rz_deg * 3.14 / 180

-- 가변 웨이포인트 설정
set_global_variable("V_X", v_x)
set_global_variable("V_Y", v_y)
set_global_variable("V_RZ", v_rz_rad)

-- 모션 실행은 티칭 파일에서 MoveL(Variable)로 수행
